## 📖 Overview
Welcome to my digital atelier—where data science meets the craft of the old masters. This interactive portfolio showcases my journey as a Data Analyst and Artificial Intelligence student, transforming complex datasets into actionable insights and strategic decisions.

Built with a focus on immersive UI/UX, this platform features 3D canvas rendering, complex scroll animations, and a responsive glassmorphism aesthetic.

## 🚀 Featured Works in The Atelier
1. **Startup Success Predictor:** An end-to-end Machine Learning web app (Random Forest) predicting startup operational viability based on historical Crunchbase data.
2. **Academic Command Center:** An executive-level analytics dashboard built for real-time monitoring of student performance metrics and critical evaluation.
3. **TikTok & Academic Procrastination:** A quantitative statistical research proposal investigating the correlation between social media consumption and academic behavior.
4. **Nanobubble Aquaculture Infographic:** An educational visual breakdown of nanobubble technology applications in the fisheries industry.

## 🛠️ Tech Stack (Portfolio Architecture)
* **Framework:** Next.js (React)
* **Animations:** Framer Motion
* **Styling:** Custom CSS & Glassmorphism UI
* **Deployment:** Vercel
